## ⚠️ 常见问题排查

### 问题 1: 点击"子应用"后没有反应
**解决方法**:
1. 检查浏览器控制台是否有错误
2. 确认子应用在 http://localhost:8081/ 正常运行
3. 查看网络面板，确认能正常加载子应用资源

### 问题 2: 子应用加载失败
**可能原因**:
- 子应用未启动
- CORS 跨域问题
- 端口不匹配

**解决方法**:
```bash
# 确保子应用正在运行
cd sub-app
npm run dev
```

### 问题 3: 样式混乱或冲突
**解决方法**:
- 已在配置中启用实验性样式隔离
- 如仍有问题，尝试刷新浏览器缓存（Ctrl+Shift+R）

### 问题 4: 端口被占用
**解决方法**:
```bash
# Windows - 停止所有 node 进程
taskkill /F /IM node.exe

# 然后重新启动
cd sub-app && npm run dev
cd main-app && npm run dev
```
