# 快速启动指南

## ✅ 应用已启动！

两个应用现在都在运行中：

### 📍 访问地址

- **主应用**: http://localhost:8080/ （已自动在浏览器打开）
- **子应用**: http://localhost:8081/ （独立访问）

### 🎯 使用步骤

1. **主应用**应该已经在浏览器中自动打开
2. 点击导航栏的 **"子应用"** 按钮
3. 你将看到子应用的内容被加载到主应用中

### 📝 开发提示

- **热更新**: 修改代码后会自动刷新
- **独立运行**: 子应用可以单独在 http://localhost:8081/ 访问
- **微前端模式**: 通过主应用访问时，子应用运行在 qiankun 容器中
- **停止服务**: 在终端按 `Ctrl+C`
